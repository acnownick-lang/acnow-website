1. Our Commitment to Accessibility

A/C Now LLC is committed to facilitating the accessibility and usability of its website, https://acnowllc.com, for all people, including those with disabilities. We strive to maintain conformance with the Web Content Accessibility Guidelines (WCAG) 2.2 Level AA standards, as defined by the World Wide Web Consortium (W3C).

2. Accessibility Features Applied

To ensure equivalent access, our site incorporates several critical accessibility features:

  • Skip to Main Content: A skip navigation link is present to bypass redundant navigation links.
  • Semantic HTML structure: Content uses logical landmark tags (<header>, <main>, <footer>, <nav>) and proper header hierarchies (H1, H2, H3).
  • Keyboard Accessibility: All interactive elements, buttons, and form inputs are navigable using standard keyboard commands (Tab, Enter, Space).
  • Aria Labels and Roles: Dynamic widgets and interactive components use ARIA tags to explain states to screen readers.
  • Color Contrast: Text colors are optimized (minimum 4.5:1 ratio) to ensure clarity for users with visual impairments.

3. WebGL Canvas Notice

Interactive 3D Canvas Accessibility

Our site features an interactive 3D Airflow Circulation Map (WebGL canvas). 3D canvas objects present real technical barriers for screen reader software, and we do not yet have a text-equivalent alternative for this specific tool. We are working to add a screen-reader-accessible summary of the airflow data it presents; in the meantime, our technicians are always available by phone to walk through the same information verbally.
